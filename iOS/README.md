#### 기타

* 상속하지 않는 클래스는 final 명시
* 함수 내부 return 은 줄바꿈으로 구분해주기
* Internal property는 타입 명시
* 클래스, 구조체 첫줄 띄우기
* computed proprety는 return을 명시한다.
* PR생성은 번갈아가면서, 머지는 항상 작성하지 않는 사람이한다.
